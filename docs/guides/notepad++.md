## Multiple Keyboard Shortcuts per Command

Other than `Scintilla commands` [^1], [use macros](https://npp-user-manual.org/docs/macros/) as a workaround for mapping multiple [keyboard shortcuts](https://npp-user-manual.org/docs/preferences/#shortcut-mapper) for the same command:

1. Record a new macro.
2. Press the existing keyboard combination for the target command.
3. Stop recording.
4. Save recorded macro and assign the new keyboard shorcut.

[^1]: > _"In the Scintilla commands tab, you can actually assign more than one shortcut to a given Scintilla command [...]"_
