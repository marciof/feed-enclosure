## Media Duration in Window Title

1. Open `Preferences`.
2. Switch to `Show settings` and `All`.
3. Open `Input / Codecs`.
4. Set `Change title according to current media` to: `$Z ($D)` [^1]

## Media Name in Screenshots

1. Open `Preferences`.
2. Switch to the `Video` tab.
3. Under `Video snapshots`, set `Prefix` to: `$Z-` [^1]

[^1]: [Format string documentation.](https://wiki.videolan.org/Documentation:Format_String/)
